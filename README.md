1. 软件简介
AeroStructCalc 是一款面向轻小型混合升力飞行器（联翼式飞艇）的结构初步设计与强度校核桌面工具。软件基于第一性原理，集成气动载荷计算、桁架式机翼结构分析、多部件重量估算、参数灵敏度分析及方案管理等功能，可帮助设计人员在概念阶段快速评估不同构型方案的结构可行性与重量特性。

应用领域：航空航天工程、无人机结构设计、飞行器概念设计。

2. 主要功能
2.1 总体参数配置
提供交互式侧边栏，支持输入起飞重量、气囊升力占比、飞行过载与安全系数、机翼展长/弦长/翼型厚度、气囊尺寸、旋翼直径与安全距离、巡航速度等关键参数。所有参数均带有推荐默认值，并内置合理性校验。

2.2 双梁结构设计
支持单梁和双梁两种构型自由切换。

每根梁可独立选择工字梁或碳管梁，几何尺寸、材料密度、许用应力由用户定义。

双梁模式下，自动按截面惯性矩比例分配总弯矩和剪力，分别校核每根梁的弯曲正应力、剪切应力，并计算各自的设计弯矩与许用弯矩。

提供加强系数，考虑局部连接与加强的影响。

2.3 翼肋与蒙皮设计
区分普通翼肋与加强翼肋，分别设置数量、厚度、填充系数、材料密度。

自动计算翼型轮廓面积、单肋净体积与总重。

蒙皮厚度、密度、弹性模量可调，自动计算翼型周长、蒙皮面积，并校核剪切稳定性（剪应力与临界屈曲应力对比）。

2.4 载荷计算
基础模式：采用均布/椭圆分布解析模型，合成气囊浮力、旋翼拉力、货舱载荷的弯矩贡献，输出根部设计弯矩、剪力及气动扭矩。

高级模式：预留 Multhopp 环量分布、可压缩性修正、突风载荷等高级算法的扩展接口。

2.5 其他重量估算
气囊系统、起落架（系数法）、载货仓（箱体表面积法）、额外质量等模块化估算。

2.6 结构重量汇总与校核
自动汇总所有部件重量，乘以全局余量系数，输出结构总重与剩余重量。

双梁模式下分前梁/后梁独立展示设计弯矩、许用弯矩、应力及通过状态。

显示各部件重量占比饼图与分解表格。

提供智能平衡建议：根据安全裕度自动提示减重或加强方向。

2.7 可视化模块
翼型几何：支持 NACA 四位数字生成或上传 .dat 文件，绘制翼型轮廓、中弧线、厚度分布，标注最大厚度和弯度。

全翼展弯矩分布：双轴展示剪力与弯矩沿全翼展的分布，根部集中标注最大弯矩和剪力值，旋翼和翼梢仅保留位置标记。

飞行器俯视轮廓：等比例绘制机翼、气囊、四旋翼及旋翼臂，纯文字标注关键尺寸，坐标轴动态适应所有部件。

2.8 方案管理与数据导出
方案保存/加载：将当前所有参数及计算结果保存为 JSON 文件，支持一键加载和应用。

批量导出：可将方案导出至指定文件夹，自动生成 JSON 参数文件和 Excel 重量报告。

方案对比：生成精简对比报告，展示总体参数、重量汇总、重量分解和强度校核的差异。

材料库管理：内置常用航空材料，支持 Excel/JSON/CSV 导入导出，提供搜索与校验功能。

计算结果导出：支持 Excel/JSON/CSV 多格式导出。

2.9 稳定性与加速服务
多层缓存：内存 LRU + 磁盘持久化，减少重复计算。

预加载机制：启动时自动加载常用库并预热核心计算函数。

会话守护：用户无操作 30 分钟后自动清理缓存并退出；侧边栏提供手动退出按钮。

参数校验：计算前对关键参数进行合理性检查，给出错误和警告提示。

性能监控：记录函数调用次数与耗时。

增强日志：文件轮转日志，多级输出。

3. 技术特点
基于第一性原理，集成桁架式机翼结构校核、多部件重量估算与方案管理功能。

支持基础/高级双模式分析，兼顾估算速度与精细度。

双梁独立校核，按刚度比分配弯矩和剪力，真实反映结构受力。

动态坐标适应，可视化模块根据实际部件尺寸自动调整显示范围。

全原创代码，无任何开源依赖，总行数超过 3000 行。

4. 运行环境与安装
4.1 硬件要求
最低配置：CPU 双核以上，内存 4 GB，硬盘 100 MB 可用空间。

推荐配置：CPU Intel i5 或更高，内存 8 GB 以上。

4.2 软件环境
操作系统：Windows 10/11、Linux、macOS

Python：3.10 及以上

依赖库（自动安装）：streamlit, numpy, matplotlib, pandas, openpyxl

4.3 安装步骤
安装 Python 3.10+。

下载项目源代码并解压，进入项目根目录。

（推荐）创建虚拟环境并激活：

bash
python -m venv .venv
.venv\Scripts\activate   # Windows
source .venv/bin/activate # Linux/macOS
安装依赖：

bash
pip install -r requirements.txt
启动软件（两种方式）：

一键启动：在 VS Code 中打开 start.py，点击右上角 ▶ 运行按钮。

命令行启动：python start.py 或 python -m streamlit run main.py。

浏览器自动打开 http://localhost:8501 即可使用。

5. 使用说明
左侧边栏：设置总体参数（起飞重量、展长、弦长、气囊尺寸等）。

选项卡切换：

🪶 主梁：选择单梁/双梁，输入截面尺寸和材料。

🪶 翼肋：设置普通肋与加强肋的数量、厚度、材料。

🛡️ 蒙皮：输入蒙皮厚度、密度、弹性模量等。

🎈 其他重量：输入气囊质量、起落架系数、货舱尺寸等。

📈 可视化：查看翼型形状、飞行器俯视轮廓、弯矩分布图。

📊 结果汇总：点击“开始计算”，查看重量分解、强度校核、平衡建议。

方案管理：在结果页底部可保存、加载、导出、对比设计方案。
